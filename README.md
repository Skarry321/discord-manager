# Discord Manager

Professional Discord server management tool.

![Build](https://github.com/Skarry321/discord-manager/actions/workflows/build.yml/badge.svg)

## Features
- Server dashboard with real-time statistics
- Channel management (create, edit, delete, bulk operations)
- Role management with visual permission editor
- Member management (roles, kick, ban)
- Bot status tracking (online, idle, DND, voice)
- Live bot logs

## Setup
```bash
npm install
npm start
```

## Bot Token
1. Go to https://discord.com/developers/applications
2. Create an application → Bot → Copy token
3. Invite bot: [Invite Link](https://discord.com/oauth2/authorize?client_id=1521850566316392520&permissions=8&integration_type=0&scope=bot)
